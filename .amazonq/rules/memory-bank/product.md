# Product Overview

## Purpose
Personal portfolio website for Katie Young — a modern, single-page application showcasing professional work, experience, skills, and contact information. Deployed to GitHub Pages at https://cheshirekate8.github.io.

## Value Proposition
A fast, accessible, visually polished portfolio with smooth animations, dark mode support, and responsive design across all devices.

## Key Features
- **Single-Page Layout** — Hero, About, Experience, Projects, Skills, Contact sections
- **Dark Mode** — Class-based toggle with persistent preference via Tailwind's `darkMode: 'class'`
- **Smooth Animations** — Framer Motion for scroll-triggered and interactive animations
- **Scroll Progress Indicator** — Visual bar showing page scroll position
- **Filterable Projects** — Interactive project showcase with live demo links
- **Custom Blue Color Palette** — Cohesive ocean-blue theme (federal-blue through light-cyan)
- **Responsive Design** — Mobile-first, optimized for all viewport sizes
- **Accessibility** — Keyboard navigation and screen reader support

## Target Users
- Recruiters and hiring managers reviewing Katie's portfolio
- Potential collaborators exploring past work and technical skills

## Deployment
- Hosted on GitHub Pages via `gh-pages` package
- Build output: `dist/` directory
- Base URL: `https://cheshirekate8.github.io`
